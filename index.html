BNPM News

NEWS website with CRYPTO and WEATHER info

This Project is built on ReactJS, HTML, CSS, and Ruby on Rails as backend server.

Users can check news of different categories like business, general, entertainment, sports, weather, science, crypto and live news video.

Home page provides all local news, top 10 crypto currency details and current location/suburb forecast for four days.

Justin page is designed to read information or news from our backend server.User can subscribe to our news letter integrated with mailchimp.

Project is integrated with cloudinary to support images and videos uploading with news information over the backend.

Project algorithm is designed to get the most related news based on the user input as well.For testing you can search with using "apple" as keyword.

Project is designed using Material UI react components to achive native look and to maintain clean code.
